## Getting Started (Setup)

1. Run the command
```bash
git clone https://github.com/Abdessittir/code-editor.git
```
2. And Run the command
```bash
npm install
```

## Prerequisites
* GitHub flow
* HTML and CSS
* JAVASCRIPT (ES6)
* React.js
* Next.js
